
############### SETUP INSTRUCIONS ###############


 - Download this FOLDER (containing everything) from Google Drive and uncompress it (double-click the .zip file). 
 
 - Name the folder whatever you want.
 
 - Type 'chrome://extensions' into your address bar on Google Chrome. Press Enter.

 - On the very top right, turn on 'Developer mode'. You can turn this off once you load the extension.

 - Now, near the top left, select 'Load unpacked'. Select the folder that you renamed earlier, and click 'open'.

 - Follow the instructions on-screen and enjoy!
 

 If you have any issues or questions, feel free to email or message me!
